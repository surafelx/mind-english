# Page Styles

- Create styles specific to the presentation of articles.

- Utilize more specific selectors for article titles, content, and metadata.

```css
/* Specific styles for the contact form in your HTML */
section {
  max-width: 600px;
  margin: 20px auto;
  padding: 20px;
  background-color: #f5f5f5;
  border-radius: 5px;
}

section h2 {
  color: #333;
}

form label {
  display: block;
  margin-bottom: 8px;
  font-weight: bold;
  color: #333;
}

form input,
form textarea {
  width: 100%;
  padding: 10px;
  margin-bottom: 15px;
  border: 1px solid #ccc;
  border-radius: 4px;
  box-sizing: border-box;
}

form button {
  background-color: black;
  text-transform: uppercase;
  font-weight: 700;
  color: #fff;
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

form button:hover {
  background-color: #004466;
}

/* Responsive styles */
@media (max-width: 600px) {
  section {
    padding: 10px;
  }
}

```
