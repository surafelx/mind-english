### Overview

Cascading Style Sheets (CSS) stands as a fundamental technology in web development, playing a pivotal role in controlling the presentation of HTML documents. It provides a powerful and flexible mechanism for styling the visual aspects of a webpage, including layout, colors, and fonts. CSS serves as a crucial tool for web developers to create visually appealing and user-friendly websites.

### Introduction

#### Overview

- CSS (Cascading Style Sheets) is a styling language used in web development.

- It controls the presentation of HTML documents, defining the layout, colors, and fonts.

#### Purpose

- Separates document structure (HTML) from presentation, enhancing modularity.

- Allows consistent styling across multiple pages and easy maintenance.

#### Syntax

- Consists of selectors and declarations.

- Selectors target HTML elements, and declarations define style properties.

#### Cascading Nature

- Refers to the order of style rules.
- Enables inheritance and overriding of styles.
- Resolves conflicts in style declarations.

### Basic Styling

#### Selectors and Declarations

- Selectors identify HTML elements, declarations define their styles.
- Examples:

```css
h1 {
  color: blue;
}
```

```css
.class {
  font-size: 16px;
}
```

#### Box Model

- Conceptualizes HTML elements as boxes with properties: width, height, padding, border, margin.
- Influences element layout and spacing.

#### Positioning

- Controls the position of elements on a webpage.
- Properties: position, top, bottom, left, right.
- Values: static, relative, absolute, fixed.

### Advanced Styling

#### Flexbox and Grid

- Layout models for advanced page structuring.
  - **Flexbox** for one-dimensional layouts (rows or columns). 
  - **Grid** for two-dimensional layouts (rows and columns).

#### Advanced Selectors

- More specific selectors for targeted styling.
- Examples: attribute selectors ([attribute=value]), pseudo-classes (`:hover`), and pseudo-elements (`::before`).

### Responsive Design

#### Media Queries

- Technique for creating responsive designs.
- Adjust styles based on the characteristics of the device.
- Commonly used to target different screen sizes and orientations
