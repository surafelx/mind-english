# Global Styles

Apply styles to the overall layout, navigation, and common elements.
Use basic selectors to set default styles for headings, paragraphs, and links.

```css
/* Reset some default styles */
body,
h1,
h2,
h3,
p,
ul {
  margin: 0;
  padding: 0;
}

body {
  font-family: 'Arial', sans-serif;
  line-height: 1.6;
}

header {
  background-color: #333;
  color: #fff;
  text-align: center;
  padding: 10px;
}

header h1 {
  margin: 0;
  font-size: 2em;
  margin: auto;
}

header img {
  margin-right: 10px;
  border-radius: 50%;
}

nav {
  background-color: black;
  text-align: center;
  padding: 10px;
}

nav a {
  text-decoration: none;
  text-transform: uppercase;
  color: white;
  margin: 0 10px;
}

section {
  max-width: 800px;
  margin: 20px auto;
  padding: 20px;
  background-color: #f5f5f5;
  border-radius: 5px;
}

section h2 {
  color: #333;
}

section p {
  padding: 10px;
  color: #555;
}

section ul {
  margin: auto;
}

section li {
  display: inline;
  padding: auto;
  text-align: center;
}

section a {
  text-decoration: none;
  color: #0077cc; /* Set your desired link color */
  transition: color 0.3s ease-in-out;
}

section a:hover {
  color: #004466; /* Change the color on hover */
}

section img {
  display: block;
  margin: 20px auto;
  border-radius: 10px;
}

section h3 {
  color: #333;
}

ul {
  list-style: none;
  padding: 0;
}

ul li {
  margin-bottom: 10px;
}

footer {
  background-color: #333;
  color: #fff;
  text-align: center;
  padding: 10px;
  position: fixed;
  bottom: 0;
  width: 100%;
}

/* Responsive styles */
@media (max-width: 600px) {
  header h1 {
    font-size: 1.2em;
  }

  section {
    padding: 10px;
  }
}

```
