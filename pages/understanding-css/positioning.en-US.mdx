# Positioning

import { Steps } from "nextra/components";

## Overview

Positioning in CSS determines how elements are placed on a webpage. It includes properties like position, top, right, bottom, and left. Understanding positioning is crucial for creating complex layouts and arranging elements precisely.

## Position Property Values

<Steps>

### Static

- Default positioning.
- Elements are positioned according to the normal flow of the document.

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <style>
      .static-positioning-example {
        width: 200px;
        height: 100px;
        background-color: lightblue;
        position: static;
        border: 2px solid darkblue;
        margin: 20px;
        padding: 10px;
      }
    </style>
    <title>Static Positioning Example</title>
  </head>
  <body>
    <div class="static-positioning-example">
      <p>This is a static positioned element.</p>
    </div>

    <p>This is some other content in the document flow.</p>
  </body>
</html>
```

### Relative

- Positioned relative to its normal position.
- Allows using top, right, bottom, and left properties for adjustment.

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <style>
      .relative-positioning-example {
        width: 200px;
        height: 100px;
        background-color: lightcoral;
        position: relative;
        top: 20px;
        left: 30px;
        border: 2px solid darkred;
        margin: 20px;
        padding: 10px;
      }
    </style>
    <title>Relative Positioning Example</title>
  </head>
  <body>
    <div class="relative-positioning-example">
      <p>This is a relatively positioned element.</p>
    </div>

    <p>This is some other content in the document flow.</p>
  </body>
</html>
```

### Absolute

- Removed from the normal flow of the document.
- Positioned relative to the nearest positioned ancestor (or the initial containing block if none).

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <style>
      .absolute-positioning-example {
        position: relative;
        width: 300px;
        height: 150px;
        margin: 20px;
        background-color: lightgreen;
      }

      .absolute-child {
        position: absolute;
        top: 20px;
        left: 30px;
        width: 150px;
        height: 75px;
        background-color: darkgreen;
      }
    </style>
    <title>Absolute Positioning Example</title>
  </head>
  <body>
    <div class="absolute-positioning-example">
      <div class="absolute-child">
        <p>This is absolutely positioned.</p>
      </div>
      <p>This content is part of the relative positioned container.</p>
    </div>

    <p>This is some other content in the document flow.</p>
  </body>
</html>
```

### Fixed

- Positioned relative to the viewport.
- Remains fixed even when the page is scrolled.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <style>
    .fixed-positioning-example {
      position: fixed;
      top: 20px;
      right: 20px;
      width: 200px;
      height: 100px;
      background-color: lightcoral;
    }
  </style>
  <title>Fixed Positioning Example</title>
</head>
<body>

  <div class="fixed-positioning-example">
    <p>This is a fixed positioned element.</p>
  </div>

  <p>This is some other content in the document flow.</p>

</body>
</html>
```

### Sticky

- Acts like `relative` positioning until an element reaches a specified point during scrolling.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <style>
    .sticky-positioning-example {
      position: sticky;
      top: 20px;
      width: 200px;
      height: 100px;
      margin: 20px;
      background-color: lightblue;
    }
  </style>
  <title>Sticky Positioning Example</title>
</head>
<body>

  <div class="sticky-positioning-example">
    <p>This is a sticky positioned element.</p>
  </div>

  <p>This is some other content in the document flow.</p>

</body>
</html>

```

</Steps>

#### Example

```css
/* Example of Positioning */
.positioned-box {
  position: relative;
  top: 20px;
  left: 50px;
}
```

In this example, the `.positioned-box` class positions an element` 20 pixels down` and `50 pixels to the right` from its normal position.
