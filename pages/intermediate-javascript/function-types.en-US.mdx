# Tags

HTML (HyperText Markup Language) is the foundation of web development, and understanding its tags is crucial for crafting web pages. In this comprehensive guide, we'll explore a variety of HTML tags, breaking down their purposes and providing examples for each.

## Anatomy of an HTML Tag

Before delving into specific tags, let's understand the basic structure of an HTML tag:

```html
<tagname attribute="value">Content</tagname>
```

- `<tagname>`: The name of the HTML tag.
- `attribute="value"`: Optional attributes that provide additional information about the tag.
- Content: The content enclosed by the opening and closing tags.

## Essential HTML Tags

### `<html>`

The `<html>` tag wraps the entire HTML document and serves as its root element.

```html
<!DOCTYPE html>
<html lang="en">
  <!-- HTML content goes here -->
</html>
```

### `<head>`

The `<head>` tag contains meta-information about the HTML document, such as title, character set, and linked stylesheets.

#### Example:

```html
<head>
  <meta charset="UTF-8" />
  <title>My Web Page</title>
  <!-- Additional meta tags and stylesheets -->
</head>
```

### `<title>`
The `<title>` tag defines the title of the HTML document, displayed on the browser tab.

#### Example:

```html
<title>My Web Page</title>
```

 `<body>`
The `<body>` tag encapsulates the content of the HTML document, including text, images, and other elements.

#### Example:

```html
<body>
  <h1>Welcome to My Web Page</h1>
  <p>This is a sample HTML document.</p>
</body>
```

### `<h1>` to `<h6>`
Heading tags define headings of different levels, with `<h1>` as the highest (most important) and `<h6>` as the lowest.

#### Example:

```html
<h1>Main Heading</h1>
<h2>Subheading</h2>
```

### `<p>`
The `<p>` tag is used for paragraphs, structuring textual content.

#### Example:

```html
<p>This is a paragraph of text.</p>
```

### `<a>`
The `<a>` tag creates hyperlinks, allowing users to navigate to other pages or resources.

#### Example:

```html
<a href="https://www.example.com">Visit Example.com</a>
```

### `<img>`
The `<img>` tag embeds images in the document.

#### Example:

```html
<img src="image.jpg" alt="Description of the image" />
```

### `<ul>` and `<ol>`
These tags define unordered (bulleted) and ordered (numbered) lists, respectively.

#### Example:

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>
```

### `<li>`
The `<li>` tag represents list items within `<ul>` or `<ol>`.

#### Example:

```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
```


### `<table>`, `<tr>`, `<td>`
These tags are used for creating tables. `<table>` defines the table, `<tr>` represents a table row, and `<td>` is used for individual table cells.

#### Example:

```html
<table>
  <tr>
    <td>Row 1, Cell 1</td>
    <td>Row 1, Cell 2</td>
  </tr>
  <tr>
    <td>Row 2, Cell 1</td>
    <td>Row 2, Cell 2</td>
  </tr>
</table>
```

### `<form>`, `<input>`
The `<form>` tag encapsulates form elements, and `<input>` creates input fields within a form.

#### Example:

```html
<form action="/submit" method="post">
  <label for="username">Username:</label>
  <input type="text" id="username" name="username" required />
  <button type="submit">Submit</button>
</form>
```

### `<div>` and `<span>`
These tags are used for grouping and styling content. `<div>` represents a block-level container, and `<span>` is an inline container.

#### Example:

```html
<div style="background-color: #f0f0f0; padding: 10px;">
  This is a block-level container.
</div>

<p>This is a <span style="color: blue;">blue</span> word.</p>
```
