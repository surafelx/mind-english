### Design the Page

### Create the `contact.html` file

### Write the HTML Structure

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <!-- Document metadata -->
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Blog - My Blog</title>
  </head>
  <body>
    <!-- Your content goes here -->
  </body>
</html>
```

- `<!DOCTYPE html>` declares the document type and version of HTML.
- `<html lang="en">` sets the language of the document to English.
- `<head>` contains metadata like character set, viewport settings, and the title of the webpage.
- `<body>` is where the main content of the webpage goes.

### Create the Header Section

```html
<!-- Inside <body> after <body> tag -->
<!-- Header section containing the main title of the blog -->
<header>
  <h1>
    <span>
      <img
        height="50px"
        width="auto"
        src="images/profile.jpeg"
        alt="Surafel's photo"
      />
    </span>
    Surafel's Blog
  </h1>
</header>
```

- `<header>` is a semantic tag for the header section.
- `<h1>` is a heading tag indicating the main title of the page.
- An `<img>` tag is used for displaying an image.

### Create the Navigation Bar

```html
<!-- Inside <body> after <header> -->
<!-- Navigation section with links to other pages -->
<nav>
  <a href="index.html">Home</a>
  <a href="about.html">About</a>
  <a href="blog.html">Blog</a>
  <a href="contact.html">Contact</a>
</nav>
```

- `<nav`> is a semantic tag for the navigation section.
- `<a> `is an anchor tag creating clickable links. Update href attributes with your actual page URLs.

### Create the Main Section

```html
<!-- Inside <body> after <nav> -->
<!-- Main content section -->
<section>
  <!-- Main content section -->
  <section>
    <h2>Contact Me</h2>

    <!-- Contact form -->
    <form action="submit_contact.php" method="post">
      <!-- Name input -->
      <div>
        <label for="name">Name:</label>
        <input type="text" id="name" name="name" required />
      </div>

      <!-- Email input -->
      <div>
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required />
      </div>

      <!-- Message textarea -->
      <div>
        <label for="message">Message:</label>
        <textarea id="message" name="message" rows="4" required></textarea>
      </div>

      <!-- Submit button -->
      <button type="submit">Submit</button>
    </form>
  </section>
</section>
```

- `<section>` is a semantic tag for grouping content.
- `<article>` is used to define independent, self-contained content, like blog posts.

### Create the Links Section

```html
<!-- After the last article inside <section> -->
<!-- Links section -->
<h3>Links</h3>
<ul>
  <li>
    <a href="https://github.com/yourusername" target="_blank">GitHub</a>
  </li>
  <li>
    <a href="https://linkedin.com/in/yourusername" target="_blank">LinkedIn</a>
  </li>
  <li>
    <a href="https://t.me/yourusername" target="_blank">Telegram</a>
  </li>
</ul>
```

- `<h3>` is a heading tag for the links section.
- `<ul>` is an unordered list container.
- `<li>`is a list item.

### Create the Footer Section

```html
<!-- Inside <body> at the end -->
<!-- Footer section -->
<footer>
  <p>&copy; 2024 Your Blog Name. All rights reserved.</p>
</footer>
```

- `<footer>` is a semantic tag for the footer section.
- `<p>` is used for text content in the footer.
