## JavaScript for Form Submission

The JavaScript code enhances the contact form's functionality by making it asynchronous. Let's understand the key aspects:

### Event Listener on Form Submission

- The `DOMContentLoaded` event ensures that the script executes only when the HTML content has fully loaded.
- The `submitForm` function is invoked when the form is submitted.

### Preventing Default Submission

- The `event.preventDefault()` method prevents the default form submission behavior, allowing for custom handling.

### Fetching Form Values

- The function retrieves values from the name, email, and message fields using `document.getElementById()`.

### Displaying Form Values

- The collected form values are logged to the `console` using `console.log()`.
- This step is useful for debugging and understanding the data being submitted.

### Additional Logic (AJAX)

- A comment encourages the addition of extra logic, such as sending data to the server using AJAX. This implies that the code can be extended to perform actions like sending form data without reloading the entire page.

### Event Listener Addition

- An event listener is added to the contact form, ensuring that the submitForm function is executed when the form is submitted.
