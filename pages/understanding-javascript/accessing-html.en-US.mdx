# Accessing HTML Elements

In web development, JavaScript provides powerful tools for accessing and manipulating HTML elements, allowing you to create dynamic and interactive web pages. In this article, we'll explore various methods for accessing HTML elements using JavaScript.

## `getElementById`

The `getElementById` method allows you to retrieve an HTML element based on its unique identifier.

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Accessing HTML Elements</title>
  </head>
  <body>
    <h1 id="pageTitle">Welcome to the Website</h1>
    <p id="content">This is some content.</p>

    <script>
      // Accessing elements by ID
      let titleElement = document.getElementById("pageTitle");
      let contentElement = document.getElementById("content");

      // Manipulating elements
      titleElement.innerHTML = "New Title";
      contentElement.style.color = "blue";
    </script>
  </body>
</html>
```

In this example, we use `getElementById` to access the HTML elements with the IDs 'pageTitle' and 'content'. We then manipulate these elements by changing the inner HTML and the text color.

## getElementsByClassName

The `getElementsByClassName` method allows you to retrieve HTML elements based on their class names. Note that this method returns a collection of elements.

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Accessing HTML Elements</title>
    <style>
      .highlight {
        background-color: yellow;
      }
    </style>
  </head>
  <body>
    <p class="highlight">This paragraph is highlighted.</p>
    <p>This is a regular paragraph.</p>

    <script>
      // Accessing elements by class name
      let highlightedElements = document.getElementsByClassName("highlight");

      // Manipulating elements
      for (let i = 0; i < highlightedElements.length; i++) {
        highlightedElements[i].style.fontWeight = "bold";
      }
    </script>
  </body>
</html>
```

Here, we use `getElementsByClassName` to access HTML elements with the class name 'highlight'. We then loop through the collection and apply styles to each element.

## querySelector

The `querySelector` method allows you to select HTML elements using CSS-style selectors.

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Accessing HTML Elements</title>
    <style>
      .important {
        color: red;
      }
    </style>
  </head>
  <body>
    <p class="important">This is an important message.</p>
    <p>This is a regular paragraph.</p>

    <script>
      // Accessing elements using querySelector
      let importantElement = document.querySelector(".important");

      // Manipulating elements
      importantElement.innerHTML = "Updated Message";
    </script>
  </body>
</html>
```

In this example, `querySelector` is used to select the HTML element with the class `'important'`, and its content is then updated.

These are just a few examples of how JavaScript can be used to access HTML elements. Understanding these methods opens up possibilities for dynamic content manipulation and interaction on your web pages. As you continue learning, you'll discover more advanced techniques for working with the Document Object Model (DOM) using JavaScript. Happy coding!
