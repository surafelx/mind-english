# HTML Forms

Forms are interactive elements that enable users to submit data to a server. HTML provides a range of form-related elements, such as `<form>`, `<input>`, `<textarea>`, `<select>`, and `<button>`.

## Basic Form Structure

A simple form includes a set of input fields and a submit button. Here's a minimal example:

```html
<form action="/submit" method="post">
  <label for="username">Username:</label>
  <input type="text" id="username" name="username" required />

  <label for="password">Password:</label>
  <input type="password" id="password" name="password" required />

  <button type="submit">Submit</button>
</form>
```

In this form:

- The `action` attribute defines the URL where the form data will be sent.
- The `method` attribute specifies the HTTP method (e.g., GET or POST).

## Textareas and Select Menus

Besides text inputs, forms can include textareas for longer responses and select menus for choosing from predefined options:

```html
<form action="/submit" method="post">
  <label for="feedback">Your Feedback:</label>
  <textarea id="feedback" name="feedback" required></textarea>

  <label for="rating">Rating:</label>
  <select id="rating" name="rating">
    <option value="5">Excellent</option>
    <option value="4">Very Good</option>
    <option value="3">Good</option>
    <option value="2">Fair</option>
    <option value="1">Poor</option>
  </select>

  <button type="submit">Submit</button>
</form>
```

## Form Validation and Attributes

HTML5 introduces attributes like required for mandatory fields and various input types (e.g., email, number) for specialized data. Example:

```html
<form action="/submit" method="post">
  <label for="email">Your Email:</label>
  <input type="email" id="email" name="email" required />

  <label for="quantity">Quantity:</label>
  <input
    type="number"
    id="quantity"
    name="quantity"
    min="1"
    max="10"
    required
  />

  <button type="submit">Submit</button>
</form>
```
