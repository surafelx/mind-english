# Semantic HTML Elements

HTML isn't just about structure; it's also about meaning. Semantic elements add context and clarity:

```html
<header>
  <h1>Website Header</h1>
</header>

<main>
  <article>
    <h2>Article Title</h2>
    <p>Article content goes here.</p>
  </article>
</main>

<footer>
  <p>Copyright Â© 2023</p>
</footer>
```
