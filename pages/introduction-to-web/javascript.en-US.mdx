## JavaScript



JavaScript is the dynamic force behind web development, enabling interactivity
and responsiveness. It allows you to manipulate HTML and CSS in real-time,
responding to user actions and creating a more engaging user experience.

JavaScript can be embedded directly into HTML or included in separate files.
Here's a glimpse of JavaScript in action:

```js filename="scripts.js"
// JavaScript in HTML
<script>
    // Function to display a greeting
    function greet() {
        alert('Hello, welcome to our website!');
    }
</script>

// JavaScript in an external file (example.js)
// Function to change the background color
function changeColor() {
    document.body.style.backgroundColor = 'lightblue';
}
```

- `function greet()`: Defines a JavaScript function to display a greeting.
- `function changeColor()`: Defines a function to change the background color.
