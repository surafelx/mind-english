## CSS


CSS (Cascading Style Sheets) is the artist's palette, adding style and visual
appeal to the web pages created with HTML. It allows you to control the layout,
colors, fonts, and overall presentation of your content.

CSS operates by selecting HTML elements and applying styles to them. Let's
consider a simple CSS example:

```css filename="styles.css"
body {
  font-family: 'Arial', sans-serif;
  background-color: #f0f0f0;
}

h1 {
  color: #333;
}

p {
  font-size: 16px;
  line-height: 1.5;
}
```

- `body`: Selects the entire body of the HTML document.
- `h1`, `p`: Selects specific HTML elements (headings and paragraphs).
- `font-family`, `background-color`, `color`, `font-size`, `line-height`:
  Various CSS properties for styling.
